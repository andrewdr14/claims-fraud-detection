Claims Fraud Detection System

This Python-MySQL-based system detects fraudulent insurance claims using rule-based logic and machine learning models.

Installation & Setup

To install dependencies, run the following:

pip install pandas numpy scikit-learn sqlalchemy pymysql flask

Database Schema

Column Type Description
claim_id INT (Primary Key) Unique claim identifier
claim_amount DECIMAL(10,2) Amount claimed

Fraud Detection Logic

def detect_fraud(claim_amount, claim_history_count): return claim_amount > 10000 or claim_history_count > 5

API Usage

Run the Flask API:

python flask_app.py
